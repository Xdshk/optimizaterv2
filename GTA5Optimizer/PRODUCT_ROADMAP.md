## 🔧 Что нужно сделать для продаваемого продукта

### 1. КРИТИЧЕСКИ ВАЖНО: GPU-мониторинг и FPS

Сейчас GPU Usage и FPS всегда показывают 0. Для полноценного продукта это обязательно:

**Вариант A: LibreHardwareMonitor (бесплатно)**
- Добавить NuGet `LibreHardwareMonitorLib`
- Получить GPU Usage, температуру, частоты, VRAM
- Работает с NVIDIA и AMD
- Интегрировать в `PerformanceMonitor.cs`

**Вариант B: NVIDIA NVML (для NVIDIA)**
- Использовать `nvidia-smi` через Process
- Или NuGet `NVIDIAGPU`

**Вариант C: MSI Afterburner / RTSS SDK**
- Для реального FPS через shared memory
- Промышленный стандарт

### 5. Функциональность

- **Применение от имени администратора** (manifest с requireAdministrator)
- **Автоочистка Majestic RP кэша** (удаление старых файлов в папке cache)
- **Очистка DNS cache** (`ipconfig /flushdns`)
- **Отключение Windows Update** во время игры
- **Приоритет сети для GTA** (QoS policies)
- **Экспорт/импорт профилей** (JSON файлы)
- **Горячие клавиши** (например Ctrl+Alt+O для оптимизации)
- **Турбо-режим** — одноразовая очистка всего перед запуском игры

### 6. Безопасность и надёжность

- **Создание точки восстановления** — доработать обработку ошибок (сейчас powershell может не сработать)
- **Валидация всех операций** — проверка результата каждого изменения
- **Rollback по таймеру** — автоматический откат через N минут после оптимизации
- **Защита от повторного запуска** (Mutex)
- **Проверка целостности файлов GTA** (опционально)
