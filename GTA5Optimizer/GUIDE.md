# GTA5 Optimizer — Инструкция по запуску

## ⚡ Быстрый запуск

### Способ 1: Запуск exe напрямую

```
Откройте папку:
  src\GTA5Optimizer.UI\bin\Debug\net8.0-windows\

Запустите:
  GTA5Optimizer.exe

Правый клик → Запуск от имени администратора
```

### Способ 2: Запуск через консоль

```bash
cd src\GTA5Optimizer.UI\bin\Debug\net8.0-windows
GTA5Optimizer.exe
```

### Способ 3: Запуск из исходников

```bash
dotnet run --project src/GTA5Optimizer.UI/GTA5Optimizer.UI.csproj
```

---

## 📋 Системные требования

| Параметр | Требование |
|----------|------------|
| ОС | Windows 10/11 x64 |
| .NET | 8.0 Desktop Runtime (или 10.0 SDK) |
| Права | **Администратор** (обязательно!) |

> ⚠️ **Важно:** Запускайте от имени администратора, иначе оптимизация не сработает!

## 🖥 Интерфейс

### Заголовок окна

```
[GTA5 OPTIMIZER] [Majestic RP]     ● статус    [─] [✕]
```

- **[─]** — свернуть окно
- **[✕]** — свернуть в трей (приложение продолжает работать)
- Окно можно перетаскивать за заголовок

### Вкладки

| Вкладка | Описание |
|---------|----------|
| **Оптимизация** | Главный экран: профиль, кнопки, метрики |
| **Мониторинг** | Детальные метрики, анализ узких мест |
| **Логи** | Журнал всех действий |
| **Настройки** | Профиль, автооптимизация, оверлей, автозапуск |

---

## 🚀 Вкладка «Оптимизация»

1. **Выберите профиль:**
   - Everyday Mode — минимальные изменения
   - **RP Mode** — рекомендуется для RP
   - Massive Online — для 100+ игроков
   - Maximum FPS — максимальная производительность

2. **Нажмите [⚡ ОПТИМИЗИРОВАТЬ]**

3. **Смотрите метрики** — FPS, CPU, GPU, RAM обновляются каждые 2 сек

---

## 📊 Вкладка «Мониторинг»

Детальная информация:
- CPU: загрузка, игра, температура
- RAM: использовано, standby, игра
- Диск: чтение, запись
- Сеть: пинг

**Анализ узких мест** — автоматически находит проблемы и даёт рекомендации.

---

## 📝 Вкладка «Логи»

Все действия записываются в журнал:

```
18:09:22  Information  MEMORY  Автоочистка памяти: RAM 93,8% > 85%
18:09:54  Information  MEMORY  Автоочистка памяти: RAM 93,7% > 85%
```

**Цвета уровней:**
- 🔵 Cyan — Information
- 🟡 Amber — Warning
- 🔴 Red — Error

Логи сохраняются в: `%LOCALAPPDATA%\GTA5Optimizer\logs\optimization.log`

---

## ⚙ Вкладка «Настройки»

- **Профиль по умолчанию** — какой профиль использовать при запуске
- **Автооптимизация** — автоматическая очистка памяти при RAM > 85%
- **Оверлей в игре** — показывает FPS, CPU, GPU поверх игры
- **Автозапуск с Windows** — запускать при входе в систему
- **Горячая клавиша** — Ctrl+Alt+O для быстрой оптимизации

## ❓ Решение проблем

### Приложение не запускается

**Ошибка: "Требуется .NET Runtime"**
```
Скачайте: https://dotnet.microsoft.com/download/dotnet/8.0
Выберите: ".NET Desktop Runtime 8.0"
```

**Ошибка: "Отказано в доступе"**
```
Запустите от имени администратора
```

**Ошибка: "Файл заблокирован"**
```
Закройте все процессы GTA5Optimizer в диспетчере задач
```

### Метрики GPU и FPS показывают 0

Это известное ограничение. Для реальных значений нужна интеграция с LibreHardwareMonitor (см. PRODUCT_ROADMAP.md).

### Оптимизация не работает

```
1. Запустите от имени администратора
2. Проверьте логи: %LOCALAPPDATA%\GTA5Optimizer\logs\
```
